# Django app for most website functions.
```

 1.  views.py - game views for list, edit, and create. also future game and predict functions. 
 also get score, spread, and other data related tasks.

 2.  models.py - main game model, stores, game id, scores, spread, everything.

 3.  urls.py - urls for list, edit, and create views.  also get score url.

 4.  templates - html and jinja templates for list, edit, and create views.

 5.  static - css, js, team logos, and profile pics
```
